# Shopsify_Dashboard
This project presents an interactive sales dashboard built using Power BI to analyze Shopify store data. The dataset was sourced from Microsoft Excel and transformed to generate meaningful business insights.
The dashboard provides a comprehensive overview of key performance metrics such as total sales, revenue trends, top-selling products, and customer purchasing behavior. Basic DAX (Data Analysis Expressions) queries were used to create calculated measures and enhance data analysis capabilities.

Key Features:
1. Sales performance analysis (daily, monthly trends)
2. Top products and category insights
3. Customer behavior and purchase patterns
3. Interactive filters and slicers for dynamic exploration
 
 Tools & Technologies Used:
1. Power BI (Dashboard & Visualization)
2. Microsoft Excel (Data Source)
3. Basic DAX (Data Analysis Expressions)
